# Floating Island

A React + Three.js 3D experimentation project exploring interactive 3D graphics and web-based visualization.

## Overview

This project is an experimental playground for learning and testing:
- **Three.js** - 3D JavaScript library for WebGL rendering
- **React Three Fiber** - React renderer for Three.js scenes
- **React Three Drei** - Useful helper components and utilities for R3F
- **Post-processing Effects** - Visual enhancements through post-processing

## Tech Stack

- **React** ^18.2.0 - UI library
- **Three.js** 0.142.0 - 3D graphics library
- **@react-three/fiber** 8.0.26 - React renderer for Three.js
- **@react-three/drei** 9.14.3 - R3F helpers and components
- **@react-three/postprocessing** 2.4.4 - Post-processing effects
